A Few words about me

I've always been into understanding systems and navigating my way around computers since I was a child. Programming has been a side hobby for me to undertake along with a few others that always had to do with optimally producing something.

I'm a quick worker and highly focused when I arrive at a problem to figure out all the multiple ways the problem could be fixed, and this presented strongly in my coding endeavours.

Specialized in

Design Analyst

My first instance of programming development started off with design rules and using a canvas to formulate designs and that's where I began Object-Oriented Programming. I worked with genetic algorithms, neural networks and physics engines to help me formulate results in my design projects.

Game Development

I've been developing games initially with the Processing language and then began using C# with Unity3D as my main game engine I use. I've also used Java to develop smaller gaming applications. This has extended into the use of augmented reality.

Virtual / Augmented Reality

As a continuation of game development, I've used the HTCVive and the Oculus Rift for testing particular virtual environments for experimental purposes again with C# and Unity3D libraries.
